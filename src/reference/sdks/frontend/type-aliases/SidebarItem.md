# Type Alias: SidebarItem

## Type declaration

### setCount()

> **setCount**: (`count`) => `void`

#### Parameters

| Parameter | Type | Description |
| ------ | ------ | ------ |
| `count` | `number` | The number to display in the badge. A value of 0 will hide the badge. |

`void`
